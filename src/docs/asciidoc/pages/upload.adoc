=== Upload file with different media type

Upload a file with other than text/plain media type, will give proper error message. API will handle files carefully regardless of the extension.

NOTE: User can upload any file by renaming its extension to *.txt*, but API will carefully examine the content of the file and respond with proper error message.

=== Unauthorized access

If an unauthorized user tries to upload a file, then API will return proper error message.

CAUTION: Unauthorized user can be a registered user in the system, but may not have the necessary permissions to access the endpoint.
